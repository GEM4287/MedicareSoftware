# MedicareSoftware

This Project is done by Ratnesh and Gaurav.
Medicare Plus plus is a modern doctors and medicine consultancy
software, that allows you to consult with your nearby doctors and

This software uses Express JS, which is a framework of Nodejs used
for RESTful APIs and used as a middleware for smooth usage
between different routes.
We use NodeMailer for sending mails to the patients when the
appointment is successful and whenever any new doctor registers
into the system and Bycrypt JS for hashing our passwords before
storing them into the database.
